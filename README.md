# Olá, eu sou a Louise! 👋

Minha entrada na tecnologia começou de um jeito que eu não esperava. No último ano da escola, eu não sabia o que queria seguir como profissão. Meus pais sugeriram algumas opções e acabaram escolhendo Engenharia de Software — e eu gostei.

Sou **Desenvolvedora Front-End Júnior** com formação em Engenharia de Software e pós-graduação em Desenvolvimento Full Stack. Atuo no desenvolvimento de aplicações web e mobile, com foco em interfaces em React Native e JavaScript.

Atualmente estou aprofundando meus conhecimentos através dos cursos do **Full Stack Club** e da **Danki Code**, e desenvolvendo projetos práticos para meu portfólio.
